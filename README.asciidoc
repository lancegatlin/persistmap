v2,02-Jan-2015

*UNDER CONSTRUCTION*

A partially persistent Map (http://en.wikipedia.org/wiki/Persistent_data_structure)
that reifies time. The past may be queried by changes may only be made "now" or
in the "future".

.Example
----

Type in expressions to have them evaluated.
Type :help for more information.

scala> :load consoleImports

scala> p.now.find(3).get

scala> p.now.deactivate(1).get
res7: Boolean = true

scala> p.now.find(1).get
res8: Option[String] = None

scala> p.old(t2).find(1).get
